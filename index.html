Stop showing ads to people who already have your app

A backend API that checks whether a user has your app installed before showing them an install ad — so your budget only reaches people who might actually convert.

The problem is simple — and expensive

When you run an app install campaign, a portion of the people seeing your ad already have your app. They will never install it again. Every impression served to them is wasted spend. adfilter catches them before the ad is shown.

How it works

One API call. Smarter ad decisions.

Before serving an app install ad, send a request to adfilter. We check instantly and tell you whether to show it or skip it.

End-to-end flow

Campaign running

Acts on decision
